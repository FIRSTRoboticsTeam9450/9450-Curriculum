Before writing any code, install the following tools. Work through them top to bottom.

## Required

- **[NI FRC Game Tools](https://www.ni.com/en/support/downloads/drivers/download.frc-game-tools.html)** — National Instruments package that installs the Driver Station and roboRIO Imaging Tool. Required to drive the robot.
- **[WPILib (VS Code)](https://github.com/wpilibsuite/allwpilib/releases)** — The FRC development environment. Installs a dedicated copy of VS Code with all Java robot libraries preconfigured.
- **[AdvantageScope](https://github.com/Mechanical-Advantage/AdvantageScope)** — Robot telemetry and log viewer for debugging and tuning. (Also bundled with recent WPILib installs.)
- **Phoenix Tuner** — Configures and diagnoses CTRE devices (motor controllers, CANcoders, Pigeon). Install from the **Microsoft Store** (search "Phoenix Tuner X").
- **[Limelight](https://limelightvision.io/pages/downloads)** — Software and firmware tools for the Limelight smart camera used for vision targeting.
